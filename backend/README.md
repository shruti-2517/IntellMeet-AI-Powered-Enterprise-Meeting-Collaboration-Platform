# Meeting Intelligence Platform - Backend

Backend server for the AI Meeting Intelligence Platform using Node.js and Express with JavaScript.

## Features
- RESTful API endpoints
- Real-time communication with Socket.io
- MongoDB database integration
- JWT authentication
- Redis caching

## Quick Start

```bash
npm install
npm run dev
```

Server will run on `http://localhost:5000`

## Environment Variables
Copy `.env.example` to `.env` and update the values.
